# ImageAugmentor

### Run the application

### Clone Repository

~~~
git clone https://github.com/FurkanTahaBademci/ImageAugmentor.git
~~~~~~~~

### Start Code

~~~
python main.py --input <input-images> --output <output-images> --type <image-type> 
~~~~~~~~
